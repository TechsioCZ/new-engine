# Standard Utilities (`@techsio/std`)

Canonical home for framework-agnostic micro-utilities shared across workspaces.

- Keep modules dependency-free and domain-free.
- Add tests for every exported behavior and edge case.
- Preserve existing semantics when replacing local helpers.
- Import explicit subpaths; no package-root barrel.
- Do not add date, currency, locale, framework, browser, or Medusa behavior here.
